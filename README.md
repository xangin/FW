如何從 GitHub 下載 .bin 檔案？

在上方的表格中，左鍵點擊您需要的 .bin 檔案連結。

進入該檔案的專屬頁面後，尋找畫面右上方（檔案內容區域的右上角）的一個 「向下箭頭 (Download raw file)」 的圖示按鈕。

點擊該 「Download raw file」 按鈕，瀏覽器就會自動下載正確的 .bin 檔案。
